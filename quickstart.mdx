# Quickstart

This guide will walk you through executing your first code snippet with Runtime. You'll learn how to make API requests and interpret the results.

## Prerequisites

Before you begin, ensure you have:

<Steps>
  <Step title="Runtime server running">
    The Runtime API server should be running on `http://localhost:8081`. See the [Setup Guide](/guides/setup) for installation instructions.
    
    Verify the server is running:
    ```bash
    curl http://localhost:8081/api/execute
    ```
  </Step>

  <Step title="HTTP client">
    You can use any HTTP client like cURL, Postman, or your favorite programming language's HTTP library.
  </Step>
</Steps>

## Execute Your First Code

Let's execute a simple "Hello, World!" program in Python.

### Using cURL

<CodeGroup>
```bash cURL
curl -X POST http://localhost:8081/api/execute \
  -H "Content-Type: application/json" \
  -d '{
    "code": "print(\"Hello, World!\")",
    "language": "python"
  }'
```

```python Python
import requests

response = requests.post(
    "http://localhost:8081/api/execute",
    json={
        "code": "print('Hello, World!')",
        "language": "python"
    }
)

result = response.json()
print(f"Output: {result['data']['output']}")
print(f"Execution time: {result['data']['executionTime']}ms")
```

```javascript JavaScript
const response = await fetch('http://localhost:8081/api/execute', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    code: 'console.log("Hello, World!");',
    language: 'javascript'
  })
});

const result = await response.json();
console.log(`Output: ${result.data.output}`);
console.log(`Execution time: ${result.data.executionTime}ms`);
```

```java Java
import java.net.http.*;
import java.net.URI;

HttpClient client = HttpClient.newHttpClient();

String requestBody = """
{
  "code": "System.out.println(\\"Hello, World!\\");",
  "language": "java"
}
""";

HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("http://localhost:8081/api/execute"))
    .header("Content-Type", "application/json")
    .POST(HttpRequest.BodyPublishers.ofString(requestBody))
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());

System.out.println(response.body());
```
</CodeGroup>

### Response

You'll receive a JSON response with the execution results:

```json
{
  "success": true,
  "message": "Code Executed successfully",
  "data": {
    "output": "Hello, World!\n",
    "exitCode": 0,
    "executionTime": 1247
  }
}
```

<Note>
  The `executionTime` is measured in milliseconds and includes container startup, code execution, and cleanup time.
</Note>

## Try Different Languages

Runtime supports multiple programming languages. Here are examples for each:

<Tabs>
  <Tab title="Python">
    ```json
    {
      "code": "for i in range(5):\n    print(f'Number: {i}')",
      "language": "python"
    }
    ```
  </Tab>

  <Tab title="Java">
    ```json
    {
      "code": "public class Main {\n    public static void main(String[] args) {\n        System.out.println(\"Hello from Java!\");\n    }\n}",
      "language": "java"
    }
    ```
    <Warning>
      For Java, the class name must match the pattern in your code. The API automatically extracts the public class name.
    </Warning>
  </Tab>

  <Tab title="JavaScript">
    ```json
    {
      "code": "const numbers = [1, 2, 3, 4, 5];\nconst sum = numbers.reduce((a, b) => a + b, 0);\nconsole.log('Sum:', sum);",
      "language": "javascript"
    }
    ```
  </Tab>

  <Tab title="C">
    ```json
    {
      "code": "#include <stdio.h>\n\nint main() {\n    printf(\"Hello from C!\\n\");\n    return 0;\n}",
      "language": "c"
    }
    ```
  </Tab>

  <Tab title="C++">
    ```json
    {
      "code": "#include <iostream>\n\nint main() {\n    std::cout << \"Hello from C++!\" << std::endl;\n    return 0;\n}",
      "language": "cpp"
    }
    ```
  </Tab>
</Tabs>

## Understanding the Response

The API returns an `ApiResponse` wrapper containing the execution results:

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request was successful |
| `message` | string | Human-readable status message |
| `data` | object | The execution result (if successful) |
| `data.output` | string | Combined stdout and stderr output |
| `data.exitCode` | int | Process exit code (0 = success) |
| `data.executionTime` | long | Execution time in milliseconds |

## Error Handling

If the code fails to execute, you'll receive an error response:

```json
{
  "success": true,
  "message": "Code Executed successfully",
  "data": {
    "output": "  File \"main.py\", line 1\n    print('Hello\nSyntaxError: unterminated string literal",
    "exitCode": 0,
    "executionTime": 892
  }
}
```

<Info>
  Note that compilation and runtime errors are captured in the `output` field. The execution itself succeeds even if your code has errors.
</Info>

### Unsupported Language

```json
{
  "success": false,
  "message": "Unsupported language: ruby",
  "data": null
}
```
